# Dashboard de Análise de Vendas 📊
Este repositório contém um projeto de análise de dados estruturado em Excel/CSV, focado no monitoramento de performance comercial, gestão de inventário e produtividade de vendas.

# Visão Geral do Projeto
O objetivo deste projeto foi transformar dados brutos de transações em insights acionáveis. A estrutura permite visualizar rapidamente quais produtos são "best-sellers", o desempenho individual da equipe de vendas e a distribuição geográfica ou por categoria.

# Estrutura dos Arquivos
* Tabela Principal: Contém o registro detalhado das transações, incluindo:

    * ID do Pedido, Data e Cliente.

    * Produto, Categoria e Segmento.

    * Métricas financeiras (Preço Unitário, Quantidade, Valor Total).

    * Vendedor responsável.

* Tabela Dinâmica: Um resumo processado que agrupa os dados para facilitar a criação de gráficos e KPIs de performance mensal e por categoria.

* Assets: Elementos visuais utilizados para a construção da interface do dashboard.

# Tecnologias e Recursos Utilizados
 * Microsoft Excel: Para tratamento de dados, criação de tabelas dinâmicas e fórmulas de busca/referência.

 * Análise de Dados: Limpeza de dados (Data Cleaning) e modelagem básica.

 * Markdown: Para documentação do projeto.
